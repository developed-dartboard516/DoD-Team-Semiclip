## 📋 What is DoD-Team-Semiclip?

DoD-Team-Semiclip is a tool designed to improve your Day of Defeat experience. It works as a Meta Mod plugin that changes how player collisions behave during team gameplay. This mod allows you to pass through teammates in certain situations, reducing frustration in tight spaces and improving overall game flow.

This tool works with official and community servers that support Meta Mod and Team Semiclip plugins. It’s lightweight and easy to set up for anyone using Windows, even if you have no prior experience with mods or game plugins.

## 📥 How to Download and Install

1. **Open the release page link** in your web browser.  
2. Look for the latest release listed at the top. The release packages will be named clearly, usually with version numbers and the word "semiclip".  
3. Download the Windows version file, usually a ZIP or DLL file. It will have names like `teamsemiclip.dll` or a ZIP archive containing the plugin files.  
4. After downloading, extract the ZIP file if necessary. You should see files like:
   - `teamsemiclip.dll`  
   - `metamod.ini` or configuration files  
5. **Locate your Day of Defeat game folder.**  
   This is typically in one of these paths:
   - `C:\Program Files (x86)\Steam\steamapps\common\Day of Defeat`  
   - Or wherever you installed Day of Defeat   
6. Inside your Day of Defeat folder, find the `addons` directory. If there is no `addons` folder, create one.  
7. Copy `teamsemiclip.dll` and any other plugin files into the `addons` folder.  
8. Open or create `metamod.ini` inside the `addons` folder and add the following line:  
   ```
   win32 addons/teamsemiclip.dll
   ```  
9. Save changes and close the file.

Meta Mod is required as a base plugin loader. To check if Meta Mod is installed: find a file called `metamod.dll` in your `addons` folder.

If you do not see it:

2. Download the latest version compatible with Day of Defeat.  
3. Install Meta Mod by following the included instructions, which usually involve copying files into your Day of Defeat `addons` folder and updating `metamod.ini`.  
4. Once Meta Mod is ready, restart your game client or server.  

DoD-Team-Semiclip will now load correctly.

## 🔧 Troubleshooting Common Issues

- **Plugin not loading:**  
  Check that `teamsemiclip.dll` is in the correct folder and referenced correctly in `metamod.ini`.  

- **Meta Mod errors:**  
  Ensure Meta Mod is installed and updated. The `metamod.ini` file must list the plugins correctly.

- **Game crashes or instability:**  
  Confirm you have the right version of the plugin for your Day of Defeat version.  
  Disable other conflicting plugins if needed.

- **Can't find the Day of Defeat folder:**  
  Right-click Day of Defeat in your Steam library, select "Properties," then "Local Files," and click "Browse Local Files." This opens the correct folder.
